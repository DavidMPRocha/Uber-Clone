## Run Locally

Clone the project

```bash
  git clone https://github.com/DavidMPRocha/Uber-Clone.git
```

Go to the project directory

```bash
  cd Uber-Clone
```

Install dependencies

```bash
  npm install
```
Or
```bash
  yarn
```

Create file .env
```bash
  GOOGLE_MAPS_APIKEY=NEW_APIKEY
```

Start project

```bash
  expo start
```
